# CMC–Alginate Hydrogel

## Recipe
For 100mL
- 3 wt% CMC in distilled water 80mL (vigorous stirring).
- 2 wt% alginate 
- 5 wt% glycerol 
1–2 h hydration before printing.
Calcium chloride bath 2 wt%

## Use
This formulation was used for slicer calibration.

## Notes
This hydrogel provided the best balance between viscosity and printability during the calibration phase.

See attached images and videos for printing results.
